Welcome to the sample site for the first milestone for this minute.

The purpose of this milestone is accomplishing two goals:

  • practice and utilize all learnt concepts so far in both HTML and CSS
  • use this site as a start point to the second, and final milestone of this minute

Although additional features and functionalities will be added later on, especially when incorporating JavaScript, this milestone should be an opportunity to explore and experiment with new and old concepts. It is a useful reminder to take a look at either this site, documentation, or even searching throughout the web for influence. If it catches your eye, or always wanted to find out how it is done, learning from online sources is always a great method of expanding your knowledge and experience.

While it is great to try things out, keep in mind that this site should meet the minimum requirements, more details on this can be found on the README located at the root directory of this milestone. Furthermore, this site should be pleasant for those browsing it. That is to say, imagine someone else had made the site, is it intuitive to use? Is the info they provide grouped in a reasonable manner? Remember, developers have a bias, and it is critical to acknowledge them when building anything for other users.
